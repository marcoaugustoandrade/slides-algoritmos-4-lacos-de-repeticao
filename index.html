Enquanto


	enquanto (condicao){
		// Faça alguma coisa!
	}
Enquanto uma condição for verdadeira repita.

Enquanto


	inteiro contador = 10
	enquanto (contador > 0)
	{
		escreva ("Detonação em: " + contador)
		contador = contador--
	}
Enquanto o contador for for maior do que 0 repita
Observe que o contador foi decrementado em cada laço executado!

Faça-Enquanto


	faca {
		// Faça alguma coisa!
	} enquanto (condicao)
Faça alguma coisa enquanto uma condição for verdadeira repita.
Observe que pelo menos uma vez o laço será executado.
