Lógica de Programação

Estruturas de repetição

Prof. Marco Antonio Andrade

Sumário

  • Enquanto: repetição com teste no início
  • Faça-enquanto: repetição com teste no fim
  • Para: repetição com variável de controle

Enquanto


	enquanto (condicao){
		// Faça alguma coisa!
	}
Enquanto uma condição for verdadeira repita.

Enquanto


	inteiro contador = 10
	enquanto (contador > 0)
	{
		escreva ("Detonação em: " + contador)
		contador = contador--
	}
Enquanto o contador for maior do que 0 repita
Observe que o contador foi decrementado em cada laço executado!

Faça-Enquanto


	faca {
		// Faça alguma coisa!
	} enquanto (condicao)
Enquanto uma condição for verdadeira repita, executando pelo menos uma vez.
Observe que pelo menos uma vez o laço será executado, ao contrário do laço enquanto.

Faça-Enquanto


	inteiro numero
	faca {
		leia(numero)
	} enquanto (numero > 0)

Para


	para(VARIAVEL_CONTADORA; CONDIÇÃO; INCREMENTO/DECREMENTO){
		// Faça alguma coisa!
	}

Para


	para(inteiro i = 0; i < 10; i++){
		escreva(i)
	}
O número de repetições é previamente conhecido pois, se i é inicializado com ZERO e a repetição irá ocorrer enquanto i < 10, serão realizadas 10 repetições.
